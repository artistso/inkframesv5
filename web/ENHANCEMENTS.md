# InkFrame Studio v5 Enhancements

This branch contains the following enhancements to InkFrame Studio:

## 1. Rulers & Guides System
- Horizontal and vertical rulers with pixel measurements
- Draggable guide lines from rulers
- Snap-to-guide functionality for precision drawing
- Toggle via Actions node

## 2. Shape Tools
- Rectangle, Ellipse, Line, and Polygon drawing tools
- Shift modifier for constrained proportions
- Filled shapes with current color and opacity
- Access via new Shapes branch under Tools node

## 3. Transform Tool
- Scale, rotate, and flip selections
- 8 transform handles (corners + edges)
- Rotation handle with live preview
- Flip horizontal/vertical quick actions

## 4. Keyboard Shortcuts System
- Comprehensive shortcuts for all major actions
- B: cycle brushes, []: brush size, U: undo, etc.
- Visual shortcut reference in Help panel
- Input field awareness (no shortcuts while typing)

## 5. Canvas Background Patterns
- Grid, dots, isometric, triangular, and perspective patterns
- Customizable opacity and spacing
- Toggle with 'G' key or Actions node
- Patterns are workspace guides only (not exported)
