InkFrame 0.5.0 brings fluid native Kotlin S Pen drawing into the complete Glass Horizon studio while preserving the canvas, perimeter and circular timelines, layers, projects, animation, and exports. Native strokes are now bound to the correct frame, layer, canvas geometry, and brush state. This update adds brush-size and eraser hover preview, a frame/layer context HUD, exact 1:1 inspection, improved pan/zoom, and a geometry-only canvas navigator. The Android release now targets API 36.
