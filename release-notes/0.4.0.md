### Brush Engine V2 studio system

- Added Profile Recovery with known-good locking, recent-change inspection, and deterministic restoration.
- Preserved the bounded Brush Profile History from the public mainline, including tuning Undo/Redo, protected locks, exact-state restoration, A/B comparison, and preset recovery.
- Added six named creative brush identities, the non-destructive Identity Mixer, Brush Match, and four-stroke Brush Signature.
- Added transient A/B preview paths, Quick Access pinning, and guarded Apply + Lock workflows without touching artwork history.

### Circular Canvas and Glass Horizon

- Added a per-project Square/Circle canvas mode with shape-aware drawing boundaries, thumbnails, autosave, archives, and transparent-corner exports.
- Added the Glass Horizon launcher icon and native Android splash presentation.
- Added an organic radial frame timeline with true circular or elliptical orbits, concentric rings, live thumbnails, navigation, playback projection, loop arcs, and S Pen scrubbing.

### Radial animation timing

- Added direct hold editing and loop handles around the radial timeline.
- Added deterministic timing rhythms and a shared timing-only Undo/Redo system.
- Added reusable timing recipes, deterministic variations, exact recipe morphs, multi-recipe phrases, persistent phrase arrangements, complete timing scores, and persistent editable score structures.
- Raised timing Undo and Redo to exactly 25 transactions.
- Added a tablet History Inspector that shows retained timing edits and can return to any reachable state through the established Undo/Redo operations.
