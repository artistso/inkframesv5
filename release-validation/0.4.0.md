# InkFrame Studio 0.4.0 validation manifest

This text-only manifest exists to trigger a mergeable pull-request validation of the consolidated 0.4.0 source tree.

The validation must cover:

- Generated debug and release Android assets
- Creator statement packaging and Studio · Steven label
- Brush Profile Recovery and preserved Brush Profile History
- Circular Canvas and Glass Horizon branding
- Radial timeline, timing structures, 25-step history, and History Inspector
- Version and Google Play metadata
- Debug APK assembly
- Signed production APK/AAB verification with the disposable CI verification key

The Play-ready tag release remains fail-closed and requires the repository’s permanent Android upload-key secrets.
